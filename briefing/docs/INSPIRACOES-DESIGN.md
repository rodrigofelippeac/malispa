# Inspirações de Design - Sites de Spa 2025/2026

## Pesquisa Realizada
Data: 29 de Janeiro de 2026
Sites analisados: 50+ exemplos de spas de luxo

## Tendências de Design Modernas

### Layout e Estrutura

#### 1. **Minimalismo e Espaço em Branco**
- Layouts limpos e arejados
- Generoso uso de whitespace
- Comunicam luxo e precisão
- Evitam poluição visual

#### 2. **Hero Section Impactante**
- Imagens full-screen de alta qualidade
- Vídeos de fundo sutis (opcional)
- CTA claro e destacado
- Mensagem simples e direta

#### 3. **Navegação Fluida**
- Menus fixos (sticky header)
- Smooth scroll entre seções
- Hamburger menu no mobile
- Acesso rápido a telefone e agendamento

### Cores e Paleta

#### Esquemas que Funcionam Bem:
1. **Neutros + Terra** (como o Mali Spa)
   - Brancos, cinzas suaves, marrons
   - Tons terrosos
   - Verde menta/sage para destaque

2. **Monocromáticos Suaves**
   - Variações de um tom
   - Elegante e sofisticado

3. **Natureza e Tranquilidade**
   - Azuis suaves
   - Verdes naturais
   - Beges e off-whites

#### ✅ Validação para Mali Spa:
- ✅ Marrom #311303 + Verde menta #e6efee = PERFEITO
- Alinhado com tendências de tons terrosos
- Transmite natureza e acolhimento
- Diferenciação (muitos usam azul)

### Tipografia

#### Tendências 2025/2026:
- **Títulos**: Serifas modernas OU sans-serif bold
- **Corpo**: Sans-serif leve e legível
- **Tamanhos**: Grandes para títulos (48px+)
- **Peso**: Light para corpo (300-400)
- **Espaçamento**: Generoso entre linhas

#### ✅ Validação para Mali Spa:
- ✅ Poppins (heading) + Inter (body) = MODERNO
- Combinação limpa e profissional
- Excelente legibilidade

### Imagens

#### O que funciona:
- **Alta qualidade**: Profissionais, bem iluminadas
- **Autênticas**: Fotos reais do spa (não stock genérico)
- **Ambiente**: Mostrar o espaço físico
- **Pessoas**: Terapeutas trabalhando (opcional)
- **Detalhes**: Close-ups de produtos, texturas
- **Natureza**: Plantas, elementos naturais

#### Formatos recomendados:
- WebP (principal)
- JPEG (fallback)
- Lazy loading
- Responsive images (srcset)

### Funcionalidades Essenciais

#### 1. **Sistema de Agendamento**
Prioridade ALTA
- Botão "Agendar" sempre visível
- Integração WhatsApp (como Mali já tem)
- Futuramente: calendário online

#### 2. **Menu de Serviços**
- Cards visuais com fotos
- Duração claramente visível
- Descrições concisas
- Preços (opcional, muitos não mostram)
- CTA individual por serviço

#### 3. **Seção "Sobre"**
- História do spa
- Filosofia e diferenciais
- Equipe (opcional)
- Foto do ambiente

#### 4. **Contato Múltiplo**
- WhatsApp (principal)
- Telefone clicável
- Instagram
- Endereço com mapa
- Horário de funcionamento

#### 5. **Social Proof**
- Depoimentos de clientes
- Avaliações (Google, Instagram)
- Quantidade de clientes atendidos
- Anos de experiência

### Seções Comuns (Ordem)

1. **Hero**
   - Imagem impactante
   - Nome do spa
   - Slogan/proposta
   - CTA principal

2. **Sobre/Introdução**
   - Quem somos
   - Nossa filosofia
   - Por que nos escolher

3. **Serviços**
   - Grid de cards
   - Visual atrativo
   - Fácil de escanear

4. **Diferenciais**
   - 3-4 pontos principais
   - Ícones + texto
   - Breve

5. **Depoimentos** (opcional)
   - Carrossel
   - Fotos de clientes (com permissão)
   - Avaliações

6. **Galeria** (opcional)
   - Fotos do ambiente
   - Lightbox

7. **Contato**
   - Múltiplas formas
   - Mapa
   - Formulário (opcional)

8. **Footer**
   - Links principais
   - Redes sociais
   - Informações legais

### Elementos Inovadores

#### Tendências Recentes:
1. **Floating WhatsApp Button**
   - Fixo no canto inferior direito
   - Animação sutil (pulse)
   - Verde característico

2. **Narrativa Holística**
   - Contar história do spa
   - Jornada do cliente
   - Conexão emocional

3. **Produtos Naturais**
   - Destaque para produtos usados
   - Ingredientes naturais
   - Sustentabilidade

4. **Pacotes/Combos**
   - Day Spa
   - Experiências combinadas
   - Gift cards

5. **Blog/Conteúdo**
   - Dicas de bem-estar
   - SEO
   - Autoridade

### Responsividade

#### Mobile-First:
- **60-70%** dos acessos são mobile
- Menu hamburger
- Botões grandes (mínimo 44x44px)
- Texto legível sem zoom
- Formulários simplificados

#### Tablet:
- Layout intermediário
- 2 colunas para serviços
- Navegação adaptada

#### Desktop:
- Experiência completa
- 3+ colunas para serviços
- Hover effects
- Mais detalhes visíveis

### Performance

#### Métricas Ideais:
- **Lighthouse**: 90+
- **LCP** (Largest Contentful Paint): < 2.5s
- **FID** (First Input Delay): < 100ms
- **CLS** (Cumulative Layout Shift): < 0.1

#### Como alcançar:
- Otimização de imagens
- Lazy loading
- Code splitting
- Minificação
- CDN (se possível)

### Acessibilidade

#### Checklist:
- [ ] Contraste adequado (WCAG AA)
- [ ] Alt text em imagens
- [ ] Labels em formulários
- [ ] Navegação por teclado
- [ ] Focus states visíveis
- [ ] Semantic HTML
- [ ] ARIA labels
